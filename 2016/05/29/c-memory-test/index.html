挖一下那些年的东东(模拟内存申请与释放C语言)

发表于 2016-05-29   |   分类于 数据结构与算法   |  

这是一个以前学数据结构时候写的demo
上面都有注释啦
用来模拟内存申请 释放 管理
先说一下原理吧

  • 首先总内存块模拟我们平时的内存
  • 申请内存时先查看是否有所需大小的连续内存块
    • 有则直接获取,并标记为已用内存
    • 没有则通过内存块合并,得到所需大小的内存
      • 若加起来都合并不了指定大小,则申请失败
      • 反之申请成功
  • 释放内存时直接去修改对应的标记就好啦
